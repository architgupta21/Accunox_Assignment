# Accuknox Django Trainee Assignment

This repository contains the solutions for the Accuknox Django Trainee assignment. It includes conclusive proofs for Django Signal behaviors (Execution flow, Threading, and Database Transactions) as well as the implementation of a custom iterable Python class.

Per the assignment instructions, this solution is structured as a standard Django project. All core logic and proofs are located within the `core/solutions.py` file.

## Prerequisites
* Python 3.x
* pip (Python package manager)
